## 使用建议

- 发给开发时优先发 `*.svg`，因为标注已经包含权限解释。
- 需要裁剪或二次编辑时，再基于 `*.png` 处理。
- 若要贴到 Confluence/Notion，优先贴 `*.png` 预览图，再附 `*.svg` 原文件链接。
